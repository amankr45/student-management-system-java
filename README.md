# Student Management System (V1)

A console-based Student Management System built using Core Java.

## Project Screenshot


## Features

* Add Student
* View Students
* Search Student by ID
* Update Student Details
* Delete Student
* Validation
* Exception Handling
* Auto Save
* Auto Load
* File Handling using Text Files
* Total Student Count

## Technologies Used

* Java
* OOP (Object-Oriented Programming)
* ArrayList
* File Handling
* Exception Handling
* Git & GitHub
