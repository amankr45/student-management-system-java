## Features

* Add Student
* View Students
* Search Student by ID
* Update Student Details
* Delete Student
* Sort Students by ID
* Sort Students by Name
* Total Student Count
* Validation
* Exception Handling
* Auto Save
* Auto Load
* File Handling using Text Files

## Technologies Used

* Java
* OOP (Object-Oriented Programming)
* ArrayList
* Collections Framework
* File Handling
* Exception Handling
* Git & GitHub
